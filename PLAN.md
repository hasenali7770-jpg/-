# خطة تحويل المشروع لمنصة كورسات قوية (مثل Udemy – لكن بهويتك)

هذا المستند يوضح **الخطة العامة** وما تم تجهيزه داخل الكود، وما سنكمله بالخطوات القادمة.

## الهدف
- الحفاظ على **نفس واجهتك الحالية** (نفس الألوان/الستايل) مع تحسين تجربة المستخدم لتكون بمستوى منصات الكورسات الكبيرة.
- إضافة بنية “منصة” حقيقية: كورسات + محاضرات + تفعيل/شراء + وصول للمحتوى.

---

## المرحلة 0 (تم تجهيزها في هذا الإصدار)
### 1) إصلاح المسارات الأساسية
- إضافة صفحة تفاصيل كورس: `/[locale]/courses/[slug]` (كانت الروابط موجودة بالكارد لكن الصفحة غير موجودة).

### 2) توسيع قاعدة البيانات (Prisma Schema)
تم إضافة الجداول الأساسية لبناء منصة:
- `Course` (مع slug + بيانات إضافية)
- `Section` و `Lesson` (هيكلة محاضرات مثل Udemy)
- `User` (هيكل أولي)
- `Enrollment` (فتح كورس لمستخدم)
- `Order` (طلب/دفع – تحضير للفيزا/ماستر)
- `ActivationCode` (كود تفعيل حقيقي مرتبط بكورس)

### 3) تفعيل الكود من السيرفر (بدل Regex)
- إضافة API: `POST /[locale]/api/activate`
  - يتحقق من الكود
  - يخلق/يجلب User عن طريق email
  - يعلّم الكود Used
  - ينشئ Enrollment

### 4) تحسين صفحة التفعيل
- إضافة حقل email
- ربطها بـ API الحقيقي
- Redirect إلى صفحة الكورس بعد النجاح

---

## المرحلة 1 (الخطوة القادمة – خلال يوم/يومين)
### A) بناء Dashboard لإدارة الكورس بشكل صحيح
داخل `/[locale]/dashboard-israa` نضيف Tabs:
1) Course Details
2) Curriculum (Sections/Lessons)
3) Publish

### B) رفع صور الغلاف (Thumbnail)
خيار مقترح للبداية:
- Cloudinary (صور)
- لاحقاً: R2/S3

الفكرة:
- API يطلع Signed Upload / أو Cloudinary signature
- الرفع يصير مباشرة للـ Storage
- نخزن الرابط بـ `Course.image`

### C) إضافة CRUD كامل للكورس
- تعديل / حذف
- عرض جميع كورسات الـ DB داخل الداشبورد

---

## المرحلة 2 (تحويله إلى “منصة مدفوعة”)
### A) الدفع بالفيزا/ماستر
أفضل مسار (اختيار واحد):
- Stripe (أفضل عالمياً)
- أو MyFatoorah / Tap / Paymob (أنسب للمنطقة)

التنفيذ:
- `POST /api/checkout`
- `POST /api/webhook`
- إنشاء `Order` بحالة pending ثم تحديثه paid
- إنشاء `Enrollment` بعد نجاح الدفع

### B) صفحة “تعلماتي” My Learning
- عرض الكورسات المفتوحة للمستخدم
- Continue Learning
- Progress بسيط

---

## المرحلة 3 (ميزات تخلي الموقع “يستحق المتابعة”)
- Wishlist
- Reviews + Ratings
- Instructor profiles
- SEO لكل كورس (OG image, metadata)
- Notifications (Email بالبداية)

---

## ملاحظات مهمة
- **لا ننصح برفع فيديوهات على السيرفر مباشرة**.
- الأفضل: Cloudflare Stream أو Mux للفيديو.
- بالبداية ممكن روابط (Vimeo/YouTube unlisted) إلى أن تجهز خدمة الفيديو.
